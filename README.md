# PDFTest

Idea of a utility to test PDFs

## Requirements

-   Module `pdf-parse`

## Usage

```bash
npm install
npm run test
```

>[!NOTE]
>To change the PDF file, change the value in lines 5 and 29 of `src/index.js`.

To corrupt a PDF, you can use this tool: https://corrupt-a-file.net/
